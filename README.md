# Ecom POC
Build a proof of concept e-commerce store, Skinet, using Angular, .Net Core and Stripe for payment processing

## Environment Details

1. .NET Core 6
2. Angular 12


## IDE Used
VS Code

## Extensions used
1. C# from Microsoft
2. C# Extensions from JosKreativ
3. Material Icon Theme
4. Nuget Package Manager and/or Nuget Gallery
5. SQLite (only for dev)
